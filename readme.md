
## API Reference

#### access api  http://localhost:3000/api/token

```http
  POST /api/token
```

| :-------- | :------- | :------------------------- |
  id | `string` | **Required** |
  user  | `string` | **Required** . username or something |
  zone          | `string` | **Required**. short zone , TH , US .etc |
